## Как запустить:
Клонировать репозиторий и установить зависимости.
```
git clone https://github.com/codelnd/test-task-apricode.git
cd test-task-apricode
npm install
```


## CLI:
```
npm run start // Запуск dev сервера
npm run build // Сборка проекта
```
